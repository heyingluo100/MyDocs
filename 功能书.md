## 文件结构总览

```
E:/个人网站/
├── content/                    # ← 文件夹即标签，拖入文件即发布
│   ├── 工作/                   #    标签「工作」
│   │   └── 报告.docx           #    Word 文件自动转为网页
│   ├── 技术笔记/               #    标签「技术笔记」
│   │   └── Vue3最佳实践.md     #    Markdown 也支持
│   └── 公告/
│       └── 欢迎.md
├── public/                     # 静态资源
├── src/
│   ├── components/             # Vue 组件（10 个）
│   ├── composables/            # Composables（8 个）
│   ├── config/invitation.js    # 邀请码系统配置（开关 + 哈希）
│   ├── views/                  # 视图页面（3 个）
│   ├── data/articles.json      # 构建生成的文章索引
│   ├── styles/main.css         # 全局样式 + 保护 CSS
│   ├── router/index.js         # 路由配置
│   ├── App.vue                 # 根组件
│   └── main.js                 # 入口文件
├── scripts/build-content.js    # 内容构建脚本
├── scripts/generate-code.js    # 邀请码生成脚本
├── PRD.md                      # 产品需求文档
├── 功能书.md                    # 本文件
├── index.html
├── vite.config.js
├── package.json
└── start.bat
```
