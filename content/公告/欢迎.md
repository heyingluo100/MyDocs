# 欢迎来到我的文档库

这是一个用于存档和分类管理文档的个人网站。你可以在这里浏览我分享的各类文章和文件。

## 功能特点

- **文件夹分类**：每个文件夹就是一个标签分类
- **拖入即发布**：把 Word 文件拖进文件夹，构建后自动展示
- **亮暗主题**：支持亮色和暗色主题切换
- **内容保护**：多层防护机制保护文档内容

## 如何添加文档

1. 在 `content/` 目录下创建一个文件夹（文件夹名就是分类标签）
2. 把 `.docx` 或 `.md` 文件拖进去
3. 运行 `npm run build` 构建
4. 部署到 GitHub Pages / Vercel
